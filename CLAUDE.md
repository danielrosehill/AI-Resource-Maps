# AI Resource Maps

## Purpose

This repository is an index of Daniel Rosehill's repositories that map out various technology and AI ecosystems — curated resource lists, project collections, and landscape overviews.

## Formatting Rules

- Organize repos by ecosystem/theme into sections
- Keep descriptions concise (1-2 sentences)
- Link directly to the relevant GitHub repos
- **Display names**: Use human-readable names (e.g. "AI Geopol Projects" not "AI-Geopol-Projects") — but ensure the name is accurate to the repo
- **Alphabetical ordering**: Sort repos alphabetically within each section
- **No per-section repo counts**: Do not add repo count badges to section headers
- **Last updated badge**: Each repo row must include a dynamic shields.io last-commit badge in an "Updated" column, using format: `![](https://img.shields.io/github/last-commit/danielrosehill/REPO-NAME?style=flat-square&label=)`
- **Header badges**: Keep top-level badges for total repo count, category count, and last-commit for this index repo
- Update the total repo count badge in the header when adding/removing repos
